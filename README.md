# Real Time AI Job Recommendation System

An end-to-end AI-powered job recommendation system that analyzes a userâ€™s resume to generate profile insights and fetch real-time job opportunities. The system uses a large language model to summarize resumes, identify skill gaps, extract job-relevant keywords, and recommend career roadmaps, while integrating live job listings from LinkedIn and Naukri via external APIs.

Built with a modular architecture supporting a Streamlit UI for interactive exploration and an MCP-based server to expose job search capabilities as reusable tools for agent and client integrations.

Tech Stack: Python, OpenAI API, Apify (LinkedIn & Naukri job scrapers), Streamlit, MCP
